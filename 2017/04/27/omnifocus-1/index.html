OmniFocus专业版初步上手指南一

发表于 2017-04-27 更新于 2017-05-08 分类于 软件 阅读次数： Valine：
本文字数： 2.4k 阅读时长 ≈ 2 分钟

前言

OmniFocus是一款时间管理应用，相信知道它的人初次接触都会被其昂贵的价格和复杂的操作给吓到。我最初用 PP 助手（学生党真心无力为了一个不知道能不能派上用场的东西花上258块钱，正版党请按 Ctrl+W ）在 iPad 上安装了一个，看了几个不知所云的自带教学就将之丢置一旁了。以前在手机上用过滴答清单，但总觉得在手机上码字和操作没有电脑爽，于是又转用 Wunderlist、MyLifeOrganized。Wunderlist 的子任务规划和任务排序做的不是很好，MLO 的 UI 比较丑，提醒设置和同步功能也比较麻烦。于是下定决心好好学习一下 OmniFocus，看看这款备受推崇的软件到底有哪些神奇之处。

安装并激活专业版

人民币玩家

  • 请按一下键盘上的 End 键，直接点击下面的赏字图标，然后掏出手机扫一扫二维码，然后没事了，谢谢啊。
  • 哦等一下，忘说怎么买了，ios 上打开 AppStore 搜索 OmniFocus，找到黑勾那个（通用版），点击258那个数字购买，安装后打开软件——设置——关于 OmniFocus——左下角128大洋升级到专业版，完事
  • 补充一下，Mac 上点击左上角 OmniFocus——许可证，输入所有者和序列号就行了，嗯，对的，你注意到了，这里是序列号，所以还要再花一次钱

ios 安装

  1. 首先，打开我们万能的x宝：

  2. 打开第一个销量最高的，评论还不错，嗯，2块钱付款，卖家发来一个 appleID，里面提供了账号密码和激活方式。

使用坚果云打造 WebDAV 服务器进行云同步

配置

  • 由于 OmniFocus 的服务器架设在国外，国内进行同步操作宛如龟速，所以极力推荐使用 WebDAV 的方式进行云同步

以下内容转载自少数派：让 OmniFocus 更快更强的 3 个实用技巧

  1. 注册一个免费的支持应用接入的本地服务，这里使用的服务是坚果云。

  2. 点击左上角「创建同步文件夹」→「新建个人同步文件夹」。

  3. 输入一个文件夹名称，这里我起名就叫 OmniFocus 。建议勾选「默认不同步到电脑」以免不小心在电脑客户端上修改同步数据。

  4. 从右上角的登录名进入「账户信息」→「安全选项」,点击「添加应用密码」

  5. 在「第三方应用管理」中新建一个应用名称

  6. 生成对应的密码：

  7. 在软件中配置同步服务器信息：

    • mac上：Command+, 打开设置，切换到同步选项卡，选择“高级（WebDAV）”，服务器地址填入 比如我的是： 然后点击“立即同步”，按照提示输入用户名和密码。用户名是你注册的坚果云的邮箱，密码是刚刚生成的应用密码。
    • ios上：打开设置——同步方法——自定（WebDAV）,在地址中填入 比如我的是： 然后点击“立即同步”，按照提示输入用户名和密码。用户名是你注册的坚果云的邮箱，密码是刚刚生成的应用密码。

问题解决

  1. 多设备配置反复弹窗要求登陆的问题

    • 我遇到的第一个问题是，在 osx 上配置完成之后，将同样的配置应用到 iPad 上时，会在同步期间反复弹出输入账户名和密码的对话框
    • 解决办法是为每一个设备都添加一个应用密码，我的最终配置如下：
      千万不要照做，这是一种sb做法。。。
    • 正确的做法是，只需要一个授权应用，在不同设备间使用同一个账号和应用密码，同步时选择保留同步数据库，至于反复弹窗，要么多输几次，要么就是网络不好，千万不要生成多个授权应用。。。，然后按照这样配置的话问题2也不会出现了。

  2. 不同设备间同步时遇到“dav.jianguoyun.com上的OmniFocus数据库的加密口令”弹窗问题

    • 参考了官方帮助文档：使用WebDAV和Omnifocus等软件同步时可能出现的错误提示和解决办法上的最后一条，删除设置的同步路径中的encrypted文件后再重新开启同步即可解决
