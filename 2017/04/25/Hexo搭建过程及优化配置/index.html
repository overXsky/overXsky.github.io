更换主题

注意：本文写于2017年，以下内容均针对NexT 5.*版本

  1. 浏览官网主题、知乎：有哪些好看的 Hexo 主题？或其他个人发布的主题，从中选择一个合适的

  2. 这里选用NexT主题，前往其发布页面下载源码zip

  3. 解压到blog目录下的themes文件夹下

  4. 打开根目录下的 _config.yml 文件，将第66行的 theme 字段值改为 hexo-theme-next-5.1.1

    使用 hexo s –debug 命令，并从浏览器访问 localhost:4000 可以进行调试访问，确保配置正确

  5. 打开主题目录下的 _config.yml 文件，找到 scheme 字段，可以通过#号注释切换主题风格为Muse/Mist/Pisces，选用 Mist 如下：

  6. 修改网站标题、副标题、描述、作者、描述、语言、时区：

    • description主要用于SEO，告诉搜索引擎一个关于站点的简单描述，通常建议在其中包含网站的关键词
    • Hexo 默认使用您电脑的时区，比如说：America/New_York, Japan, 和 UTC 。参考：时区列表。这里采用Asia/Shanghai

  7. 修改个人头像：在source文件夹下新建uploads文件夹，放入头像文件，并在主题目录下的配置文件中将avatar一行改为avatar: /uploads/avatar.jpg

新增文章

  • 使用命令hexo new filename 即可生成一个新的md文件，然后用编辑器修改即可
  • 也可以把写好的md文件直接拷贝到\source_posts</code>目录下

hexo generate 命令可以生成静态的网页文件
hexo clean 可以清除上面生成的文件

代码高亮

  • 根目录配置文件的 highlight 的 enable 值设为 true
  • 主题配置下有5种样式可选，直接修改保存即可

文章添加更新日期

  • 将主题配置文件中 post_meta 字段中 updated_at 值改为 true

修改网站图标

更多的 icon 可以查看Font Awesome，hexo默认导入该网站图标

  • 网站图标的配置在主题配置文件中，默认为 favicon.ico 文件，需要放置在网站目录下的 source 文件夹中
  • 经过实际测试可以任意更改文件名，比如我是把 favicon.png 放在了 blog/source 下

解决报错

首先遇到配置以后未生效的问题，都先clean再generate一次，还不行就重新覆盖安装插件再clean g一次，重启server即可解决

  • 修改配置文件后使用hexo server命令报错：
  • 参看了使用hexo+github搭建个人博客中的问题1得到解决方案：编辑_config.yml文件，每个配置项后面必须有一个空格!
  • 检查配置文件，发现language:zh-Hans冒号后缺少空格，改成language: zh-Hans这样即可
